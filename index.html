Mélange suggéré par Claude

En plus du calcul colorimétrique hors ligne, vous pouvez demander à Claude de proposer un mélange en s'appuyant sur ses connaissances (et une recherche web si utile) pour une teinte précise. Ceci nécessite votre propre clé API Anthropic.
Créez une clé sur console.anthropic.com (rubrique API Keys). Chaque demande consomme un petit crédit sur votre compte Anthropic (facturation à l'usage). La clé est stockée uniquement sur cet appareil.

Mélange suggéré par votre Mac

Appelle un modèle Ollama installé sur ce Mac. Gratuit, aucune donnée ne quitte l'appareil.
Ne fonctionne que sur ce Mac — ouvrez cette appli directement dans Safari sur la machine où tourne Ollama. Depuis l'iPad, ce bloc ne pourra pas le joindre (ce n'est pas un bug : localhost désigne l'appareil qui l'utilise, pas le Mac à distance).
Sur le Mac, lancez Ollama en autorisant cette appli (adresse exacte de votre appli, sans chemin ni barre oblique finale) :
OLLAMA_ORIGINS="https://votre-pseudo.github.io" ollama serve
Détails dans le README si besoin.

Synchronisation GitHub

Vos peintures et mélanges sont stockés dans un Gist GitHub privé, ce qui les synchronise automatiquement entre vos appareils. Voir le fichier README fourni pour créer le Gist et le token.
Synchronisation active. Chaque modification est envoyée automatiquement vers votre Gist GitHub.

Exporter / Importer

Sauvegardez votre stock et vos mélanges, ou restaurez-les sur un autre appareil.

Exporter

Génère un fichier .json à télécharger. Gardez-le précieusement, ou envoyez-le vous-même à un autre appareil (iCloud, mail, AirDrop...) pour l'importer là-bas.

Importer

Sélectionnez un fichier .json exporté depuis Arsenal Chromatique. Les données sont fusionnées avec celles déjà présentes (rien n'est écrasé sans confirmation) : les peintures déjà connues sont mises à jour, les nouvelles sont ajoutées, et les mélanges déjà connus ne sont pas dupliqués.
